06 — IF Node Branch Logic

✓ TRUE — snippet is not empty

  • Lead has replied to the cold email
  • snippet field has content from Gmail Thread
  • Gemini analyzes the reply text
  • Gemini classifies intent + drafts reply
  • Gmail sends the reply
  • Sheet updated: Replied + Engaged

✗ FALSE — snippet is empty

  • Lead has not replied yet
  • Goes to No Operation node
  • Sheet is not updated
  • Loop moves to next lead
  • Lead re-checked on next 30-min run
!Add a "No Operation, do nothing" node on the false branch. Search it in n8n and connect the false/bottom output of IF to it. This prevents the canvas warning about unconnected outputs.
