04 — IF Node Branch Logic

✓ YES — snippet exists

  • Lead has replied to the cold email
  • Pass snippet to AI node for intent detection
  • AI drafts a contextual reply (<100 words)
  • Gmail sends the reply
  • Sheet updated: Replied + Engaged

✗ NO — no snippet / no reply

  • Lead has not replied yet
  • Flow skips all remaining nodes
  • Sheet is not updated
  • Lead re-checked on next cron run
05 — Setup Checklist
1

Create Google Sheet

Add columns: Email, Company Name, Cold Mail Status, Reply Status. Fill in your leads. Copy the Sheet ID from the URL.

2

Connect Google Sheets credential

In n8n → Credentials → New → Google Sheets OAuth2. Authorize with your Google account. Use this credential in nodes 2 and 8.

3

Connect Gmail credential

In n8n → Credentials → New → Gmail OAuth2. Authorize with the same Google account that sent the cold emails. Use in nodes 4 and 7.

4

Add AI / LLM credential

For OpenAI: add OpenAI API Key credential. For Gemini: add Google Gemini API Key. Use in node 6.

5

Replace Sheet ID placeholders

In nodes 2 and 8, replace YOUR_GOOGLE_SHEET_ID with the actual ID from your sheet URL: docs.google.com/spreadsheets/d/SHEET_ID/edit

6

Test with one lead

Add a single test email row, manually trigger the workflow, and verify the Gmail search, AI response, and sheet update all work before enabling the cron.

7

Activate the workflow

Toggle the workflow to Active. The cron will fire every 30 minutes automatically and process any new unread replies.
